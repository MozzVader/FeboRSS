Todo lo que necesitás, nada que sobre

Diseñado para power users que quieren eficiencia sin complicaciones

📰

Vista dual

Cards con thumbnails y resumen, o vista compacta para leer rápido. Cambiá con un click o un atajo.

⏰

Auto-refresh configurable

Elegí cada cuánto se actualizan tus feeds: 5, 15, 30 o 60 minutos. O desactivalo y refrescá manualmente.

📂

Categorías + Drag & Drop

Organizá tus feeds en carpetas colapsables. Reordená con drag & drop directo en el sidebar.

🔔

Notificaciones de escritorio

Recibí alertas cuando hay artículos nuevos en los feeds que elijas. Silenciá todo con un click.

🔗

OPML Import/Export

Migrá desde cualquier lector RSS. Importá y exportá en formato OPML estándar compatible.

⚡

Reddit-ready

Manejo especial de imágenes, GIFs y galerías. Credenciales RSS integradas para evitar rate limiting.

🛑

NSFW Filtering

Marcar feeds como NSFW para excluirlos de vistas globales. Accedé individualmente cuando quieras.

💡

Atajos de teclado

Navegación completa sin mouse. j/k para moverte, s para favorito, r para refrescar, y más.

🌙

Tema claro/oscuro

Cambiá el tema con un click o con `t`. Tu preferencia se guarda automáticamente.

Listo en 4 comandos

Nada de configuraciones complicadas. Cloneá, instalá y arrancá.

Terminal 
# Clonar
$ git clone https://github.com/MozzVader/FeboRSS.git
$ cd FeboRSS

# Instalar dependencias
$ npm install

# Inicializar base de datos
$ npx prisma db push

# Arrancar
$ npm run dev
